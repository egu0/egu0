[hexo setup](https://hexo.io/docs/)

```sh
npm install hexo
npx init DIR_NAME
cd DIR_NAME
npm install
export PATH="$PATH:./node_modules/.bin"
hexo server

# Localization testing
hexo clean && hexo s
# Deployment:
hexo g -d
```
