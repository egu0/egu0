# Egu0's blog

[hexo setup](https://hexo.io/docs/)

```sh
npm install hexo
npx hexo init DIR_NAME
cd DIR_NAME
npm install
export PATH="$PATH:./node_modules/.bin"

# Start server
hexo s
# Localization testing
hexo clean && hexo s
# New post
hexo new [POST_TITLE]
```
